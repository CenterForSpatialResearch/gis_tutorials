    <article class="markdown-body entry-content" itemprop="text"><h1><a href="#exporting-3d-data-from-arcgis-to-rhino" aria-hidden="true" class="anchor" id="user-content-exporting-3d-data-from-arcgis-to-rhino"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Exporting 3D Data from ArcGIS to Rhino</h1>
<h2><a href="#visualizing-3d-data-in-arcscene" aria-hidden="true" class="anchor" id="user-content-visualizing-3d-data-in-arcscene"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Visualizing 3D Data in ArcScene</h2>
<p>Make a 3D file in ArcScene using this <a href="https://github.com/CenterForSpatialResearch/gis_tutorials/blob/master/05_Visualizing_Data_in_3D_Using_ArcScene.md">tutorial</a>.</p>
<h2><a href="#exporting-data" aria-hidden="true" class="anchor" id="user-content-exporting-data"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Exporting Data</h2>
<p>Enable the 'Data Interoperability' and '3D Analyst' extensions by going to 'Customize --&gt; Extensions' in the toolbar of either ArcCatalog or ArcScene. Failure to do so will result in the necessary tools not running
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-0.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-0.png" alt="t26-0.png" style="max-width:100%;"></a>
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-1.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-1.png" alt="t26-1.png" style="max-width:100%;"></a></p>
<p>Go to the 'layer 3D to Feature Class' tool in 'Arctoolbox --&gt; 3D Analyst tools --&gt; Conversion'
This tells arcscene that the layer is a mesh object that can be transformed to other relevant file types. Selecting the 'folder' button to the right of the 'Output Feature Class' address bar lets you change the destination and name of the output file
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-2.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-2.png" alt="t26-2.png" style="max-width:100%;"></a>
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-3.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-3.png" alt="t26-3.png" style="max-width:100%;"></a></p>
<p>Select 'Quick Export' from 'Arctoolbox --&gt; Data Interoperability tools'. Click the 'null' export type and change it to Autodesk 3ds. Select the folder in which to save the file.
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-4.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-4.png" alt="t26-4.png" style="max-width:100%;"></a>
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-5.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-5.png" alt="t26-5.png" style="max-width:100%;"></a>
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-6.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-6.png" alt="t26-6.png" style="max-width:100%;"></a></p>
<h2><a href="#importing-to-rhino" aria-hidden="true" class="anchor" id="user-content-importing-to-rhino"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Importing to Rhino</h2>
<p>'Import' the file into Rhino. You can edit the Rhino import settings if you wish, or you can just leave it default. You might need to 'Zoom Extents' to find the location of the objects.
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-7.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-7.png" alt="t26-7.png" style="max-width:100%;"></a></p>
<p>You'll notice that the faces of the objects imported from ArcScene render incorrectly so to fix that, you need to 'select all' and 'explode' the meshes. Now, all faces should be rendered properly on all sides.
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-8.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-8.png" alt="t26-8.png" style="max-width:100%;"></a>
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-9.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-9.png" alt="t26-9.png" style="max-width:100%;"></a></p>
<h2><a href="#importing-with-z-axis-intact" aria-hidden="true" class="anchor" id="user-content-importing-with-z-axis-intact"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Importing with Z axis intact</h2>
<p>If the ArcScene layers have X, Y and Z axes, i.e. floating on a contour surface or particular z axis heights, you can still import them to Rhino at those heights. Refer to <a href="https://github.com/CenterForSpatialResearch/gis_tutorials/blob/master/09_Creating_A_3D_Site_Model_In_GIS.md">this tutorial</a> for help with this. Select 'feature to 3D by attribute' from 'Arctoolbox --&gt; 3D Analyst Tools --&gt; Feature to 3D by Attribute'. This essentially allows you to set the heights at which each feature is extruded from using a particular attribute.
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-10.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-10.png" alt="t26-10.png" style="max-width:100%;"></a></p>
<p>Select the 'height field' as the 'Z axis', which would be the base heights of the features (ground elevation in this case). This process can also be used for making contour lines 3D.
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-11.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-11.png" alt="t26-11.png" style="max-width:100%;"></a></p>
<p>Afterwards, extrude to the heights of each feature by rightclicking on the layer name and going to ' Properties'. Navigate to the tab titled 'Extrusions' and check 'Extrude features in layer'. To set the 'extrusion value', click the 'calculator' to the right of 'Extrusion value or expression'.
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-12.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-12.png" alt="t26-12.png" style="max-width:100%;"></a></p>
<p>After this, repeat the process of exporting 'layer 3D to feature class' and 'importing' to Rhino.</p>
<h2><a href="#exporting-contour-lines-for-rhino" aria-hidden="true" class="anchor" id="user-content-exporting-contour-lines-for-rhino"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Exporting contour lines for Rhino</h2>
<p>Select 'export to CAD' from 'Conversion Tools --&gt; To CAD.' This tools let you export a layer or multiple layers to CAD, at which point you can then'import' to Rhino. The CAD file will already be 3D if its base heights were already set up in ArcScene.
<a href="https://github.com/tolaoniyangi/gis_tutorials/blob/master/Images/Tutorial_26/t26-13.png" target="_blank"><img src="https://github.com/tolaoniyangi/gis_tutorials/raw/master/Images/Tutorial_26/t26-13.png" alt="t26-13.png" style="max-width:100%;"></a>
